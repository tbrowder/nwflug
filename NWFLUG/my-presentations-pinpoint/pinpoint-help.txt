Usage:
  pinpoint [OPTION...] - Presentations made easy

Help Options:
  -h, --help                             Show help options
  --help-all                             Show all help options
  --help-clutter                         Show Clutter Options

Application Options:
  -m, --maximized                        Maximize without window decoration, instead
                                         of fullscreening, this is useful
                                         to enable window management when running
                                         [command=] spawned apps.
  -f, --fullscreen                       Start in fullscreen mode
  -s, --speakermode                      Show speakermode window
  -r, --rehearse                         Rehearse timings
  -o, --output=FILE                      Output presentation to FILE
                                         (formats supported: pdf)
  -c, --camera=DEVICE                    Device to use for [camera] background
  --display=DISPLAY                      X display to use
  --screen=SCREEN                        X screen to use
